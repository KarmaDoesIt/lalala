Adware

¿Qué es un adware maligno?

Un adware o programa publicitario es un tipo de programa maligno que está diseñado para automáticamente mostrar, descargar o inyectar publicidad en el sistema del usuario. Si bien el adware es relativamente inofensivo y se incluye legalmente con software gratuito, el adware malicioso puede rastrear actividad, disminuir el rendimiento del sistema y exponer a usuarios a amenazas mayores (Symantec, 2021).

A menudo los adwares se infiltran en las computadoras por medio de descargas de freeware o software gratuito, el software incluido y ventanas emergentes engañosas que engañan al usuario a hacer clic en enlaces maliciosos. A diferencia de un software publicitario legítimo, los dañinos operan sin consentimiento del usuario, mostrando publicidad invasiva y coleccionando datos personales para beneficio financiero.

¿Cómo afecta un adware a una computadora?

Una vez instalado, el adware puede causar varios problemas, incluyendo:

  • Anuncios emergentes intrusivos: Anuncios no deseados aparecen con frecuencia e interrumpen la navegación en la web.
  • Secuestro de navegador: Ciertos programas publicitarios cambian los ajustes del navegador y redirigen a sitios web sospechosos.
  • Disminución de potencia del sistema: El exceso de anuncios y procesos en segundo plano consumen memoria y la CPU.
  • Colección de datos y riesgos de privacidad: Algunos programas publicitarios rastrean los hábitos de navegación del usuario y venden datos personales a anunciantes o ciberdelincuentes (McAfee, 2022).
  • Aumento de riesgos de seguridad: Hacer clic en anuncios emergentes causados por adware puede llevar a infecciones de malware adicionales, como spyware o ransomware.

¿Cómo combatir un adware o programa publicitario?

Evitar infecciones de programas publicitarios:

  • Descargar software de fuentes confiables: Evite sitios terceros que incluyen adware con las descargas.
  • Usar herramientas de antivirus y anti-adware de reputación: Programas como Malwarebytes, AdwCleaner o Windows Defender ayudan a detectar e impedir el adware.
  • Tener cuidado con software gratuito: Siempre escoja “instalación personalizada” para desmarcar complementos no deseados.
  • Habilitar bloqueador de ventanas emergentes: Los navegadores modernos ofrecen herramientas para prevenir ventanas emergentes maliciosas.

Eliminar adware de una computadora:

  • Desinstalar programas sospechosos: Revise la lista de aplicaciones instaladas y desinstale software desconocido e indeseado.
  • Reinicie los navegadores web: Limpie los ajustes del navegador, el caché y extensiones que puedan estar vinculadas al adware.
  • Ejecute un escaneo Anti-Adware: Use herramientas de seguridad para detectar y eliminar infecciones de adware.
  • Revise los archivos de Host y la configuración de DNS: El adware malicioso puede modificar esos ajustes para controlar el tráfico de redes.
  • Realice una restauración del sistema: Si el adware persiste, es recomendado que restablezca el sistema a un estado limpio previo.

Conclusión

El adware es una amenaza cibernética persistente y muy seguido subestimada que interrumpe la experiencia del usuario y compromete su seguridad. Los usuarios pueden prevenir y eliminar efectivamente el adware de sus sistemas al practicar hábitos de navegación seguros, usar herramientas de seguridad y manejar las instalaciones de software con cuidado.
